### Get Order

Retrieves detailed information about a specific order based on its unique identifier.

This method allows clients to fetch the details of a particular order from the dYdX exchange by providing its order ID. The method sends a GET request to the `/v4/orders/{orderId}` endpoint and returns the order data.
