### Parent Subaccounts

Data feed of a parent subaccount. This feed returns the same data as the [subaccount data feed](/indexer-client/websockets/subaccounts).

A parent subaccount is a subaccount numbered between 0 and 127. Used for isolated position management by the dYdX frontend (web).
