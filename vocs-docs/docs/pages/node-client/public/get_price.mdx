### Get Price

The `Get Price` method is used to retrieve the current market price for a specified market, identified by its Clob pair `id`.

It returns the price as a `MarketPrice` object when the request is successful.

#### Method Declaration

:::code-group

```python [Python]
async def get_price(self, market_id: int) -> market_price_type.MarketPrice
```

```typescript [TypeScript]
async getPrice(marketId: number): Promise<PricesModule.QueryMarketPriceResponse>
```

```rust [Rust]
pub async fn get_price(&mut self, market_id: u32) -> Result<MarketPrice, Error>
```
